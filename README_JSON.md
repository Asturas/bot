# 🤖 Discord Bot - نظام اقتصادي متقدم (JSON Storage)

## ✨ المميزات

- 💰 نظام اقتصادي متكامل
- 🎮 4 ألعاب تفاعلية
- 📊 نظام أسهم
- 💾 تخزين بـ JSON (بدون MongoDB)
- 🌐 لوحة تحكم ويب
- 📱 34 أمر متقدم
- ✅ مجاني 100%

---

## 🚀 البدء السريع

```bash
# تثبيت المكتبات
npm install

# إنشاء ملف البيئة
cp .env.example .env

# تعديل .env وإضافة البيانات
# DISCORD_TOKEN, CLIENT_ID, GUILD_ID

# تشغيل البوت
npm start
```

---

## 📝 الأوامر الأساسية

```
/balance      - عرض الرصيد
/daily        - مكافأة يومية
/rps rock 100 - لعبة ورقة حجرة
/help         - دليل الأوامر
```
