# 🤖 Discord Bot - نظام اقتصادي متقدم

## ✨ المميزات الكاملة

### 💰 النظام الاقتصادي
- **عملة خاصة** 🏆 (العملة الرسمية)
- **محفظة شخصية** للأموال النقدية
- **حساب بنكي** لادخار الأموال
- **نظام أسهم متقدم** مع أسعار متذبذبة
- **مخزون شخصي** لتخزين الأشياء
- **تحويل أموال** بين المستخدمين
- **مكافآت يومية** مع نظام تسلسل

### 🎮 الألعاب التفاعلية
1. **🪨 ورقة حجرة مقص** - لعبة كلاسيكية مع رهانات
2. **🎲 النرد** - توقع النتيجة (عالي/منخفض، زوجي/فردي)
3. **🎰 السلوتس** - ماكينة القمار مع جاكبوت
4. **🃏 بلاك جاك** - لعبة الورق الشهيرة

### 📊 الأسهم والاستثمار
- شراء وبيع الأسهم
- تتبع أسعار الأسهم
- عرض محفظة الأسهم
- رسم بياني للأسعار التاريخية

### 👨‍💼 أوامر الإدارة (Admin Only)
- `addmoney` - إضافة أموال لمستخدم
- `removemoney` - خصم أموال من مستخدم

### 📊 لوحة التحكم الويب
- عرض الإحصائيات العامة
- لوحة المتصدرين
- قائمة الأسهم
- سجل المعاملات
- واجهة حديثة وسريعة

### 💾 قاعدة البيانات
- MongoDB للتخزين الآمن
- حفظ جميع بيانات المستخدمين
- تسجيل المعاملات
- إحصائيات اللعب

## 📝 أوامر الاقتصاد

### المحفظة والبنك
- `/balance` - عرض رصيدك
- `/bank deposit <amount>` - إيداع أموال
- `/bank withdraw <amount>` - سحب أموال
- `/bank status` - حالة حسابك البنكي
- `/daily` - مكافأة يومية

### التحويلات
- `/transfer <user> <amount> [message]` - تحويل أموال

### الأسهم
- `/stocks list` - قائمة الأسهم
- `/stocks buy <stock> <shares>` - شراء أسهم
- `/stocks sell <stock> <shares>` - بيع أسهم
- `/stocks portfolio` - عرض محفظتك

### السجلات
- `/history [limit]` - سجل المعاملات
- `/inventory` - المخزن الشخصي
- `/gamestats` - إحصائيات اللعب
- `/richest` - أغنى لاعبين

## 🚀 التثبيت والتشغيل

### المتطلبات
- Node.js 16+
- MongoDB (Atlas أو محلي)
- Discord Bot Token

### خطوات التثبيت

```bash
# 1. استنساخ المشروع
git clone https://github.com/Asturas/bot.git
cd bot

# 2. تثبيت المكتبات
npm install

# 3. إنشاء ملف البيئة
cp .env.example .env

# 4. تعديل .env وإضافة:
DISCORD_TOKEN=your_token
CLIENT_ID=your_client_id
GUILD_ID=your_server_id
MONGODB_URI=mongodb+srv://username:password@cluster.mongodb.net/bot_db

# 5. تشغيل البوت
npm start

# 6. تشغيل لوحة التحكم (في تيرمينال منفصل)
node server/server.js
```
